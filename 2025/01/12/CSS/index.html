选择器

选择器包括三种：标签选择器、类选择器、id选择器

优先级：id选择器>类选择器>标签选择器

标签选择器：重写覆盖原标签的样式

类选择器：跨标签的特性，可被多次使用

id选择器：每个标签id全局唯一，只能被使用一次

层次选择器

层次选择器包括四种：后代选择器、子选择器、相邻兄弟选择器、通用兄弟选择器

后代选择器：选择所有后代

子选择器：只选择下一代

相邻兄弟选择器：只选择下一个兄弟

通用兄弟选择器：选择以下所有兄弟

结构伪类选择器

通过位置选择：例如：last-child，nth-of-type（2）

nth-of-type(n) 表示:父容器下,某类中的第几个

nth-child(n) 表示:父容器下第几个

状态伪类选择器

属性选择器

通过条件选择：例如：href = “https://www.baidu.com“

标准文档流

HTML内所有元素可以被分为块级元素 和 行内元素

块级元素：h1-h6；div；p；ul；ol …

行内元素：a；span；img；strong …

其中,块级元素独占一行,而行内元素不独占一行

前者可以包括后者,反之,不成立

使用display可以定义一个元素的特性

display：block -> 块元素

display：inline -> 行类元素

display：inline-block ->同行显示的块元素

例如：使用inline-block将四个div排列在同一行

定位

相对定位

使得元素相对于自己原先的位置进行一定程度的偏移

移动后，原先该元素占据的位置被保留，依然存在于标准流中，只是视觉上的偏移

绝对定位

绝对定位是 : 将元素移动到当前窗口的一个指定坐标上

移动后的元素将不存在于标准流当中，这是和相对定位的重要区别

因此，可以观察到此处的父容器塌陷

fixed和absolute的区别

1、没有滚动条的情况下没有差异

2、在有滚动条的情况下，fixed定位不会随滚动条移动而移动，而absolute则会随滚动条移动
